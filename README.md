# First Game Jam 2026

This project is for [My First Jam 2026](https://itch.io/jam/my-first-jam-2026). 

## At a Glance
- Engine: Godot 4
- Language: GDScript
- Open: `project.godot`
- Run: `scenes/main.tscn`

## Getting Started

1. Install Godot GDScript verison.
- Go [here](https://godotengine.org/download/windows/)
- Ensure you're not downloading the Mono version
- Open editor
- You're done!
2. Install [git](https://git-scm.com/install/windows)
- Download and run "Git for Windows/x64 Setup" installer
- OR open Powershell and type `winget install --id Git.Git -e --source winget`
3. Clone the project
- Navigate to the [github page](https://github.com/fire00f1y/First-Game-Jam-2026)
- Click the green `Code` button and copy the text field 
- In a terminal or powershell window navigate to where you want the project by using `cd <project-parent-folder>` 
- Clone the github repo with `git clone https://github.com/fire00f1y/First-Game-Jam-2026.git` (this is the line you just copied)
4. Open project in the editor
- Load godot

### Optionally
Some further setup you can do, but is not required for initial setup:
1. Install an editor
- Choose [VSCode](https://code.visualstudio.com/) or [Jetbrains Rider](https://www.jetbrains.com/rider/). I use and prefer Rider. Both are free.
- Download and use the installer.
- Open Godot (after installing the editor)
- Go to `Editor -> Editor Settings` and click the "Advanced Settings" toggle.
- Go to `Text Editor -> External` and put in the Exec Path of your editor, wherever you installed it
2. Log into github and ensure you can `git push` to the repository
- Make sure you have an account
- Ask me to add your account as a collaborator
- After I do, edit this `README.md` file with something small
- Add the change by typing `git add .` in the project directory in your terminal
- Commit the change by typing `git commit -m "I made a super cool change"` or whatever other message
- Push the change by typing `git push` and it should work without giving you an issue

**Side Note:** if you're interested, Rider has a [good walkthrough](https://www.jetbrains.com/guide/gamedev/tutorials/rider-godot-pong/) of making a simple game using Godot and Rider.
